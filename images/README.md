# Images Library

This folder contains all screenshots, architecture diagrams, data model illustrations, Snowflake implementation screenshots, dashboards, and project visuals for the RideNow Enterprise Data Platform.

---

## Folder Structure

| Folder | Description |
|---------|-------------|
| architecture | High-level architecture, low-level architecture, data flow, and Medallion Architecture diagrams |
| database | Snowflake database, schemas, warehouses, roles, and setup screenshots |
| data_model | ER Diagram, Star Schema, DIM/FACT relationships, and SCD illustrations |
| snowflake | File formats, stages, COPY INTO, streams, tasks, query profile, and other Snowflake features |
| dashboards | Power BI dashboard screenshots |
| github | Repository structure and GitHub project screenshots |
| project | Roadmap, technology stack, timeline, and project planning visuals |

---

## Naming Convention

All image files follow the naming convention:

01_Name.png
02_Name.png
03_Name.png

Benefits:

- Easy to maintain
- Logical ordering
- Consistent naming across the repository

---

## Image Guidelines

- Use PNG format whenever possible.
- Use descriptive file names.
- Number images sequentially within each folder.
- Keep screenshots clear and high resolution.
- Remove sensitive information before uploading.

---

## Purpose

These images support the project documentation and help explain the architecture, implementation, and overall project workflow.
